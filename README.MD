## Preview the app :

This is app is deployed to Heroku, you can preview it on this [Link](https://dev-ghazwan-aliesh.herokuapp.com)

## Installation :

```
npm install
```

```
cd client && npm install
```

or

```
yarn install
```

```
cd client && yarn install
```

## Running the app :

```
npm -g install nodemon
```

```
npm run dev
```

or

```
yarn global add nodemon
```

```
yarn run dev
```
